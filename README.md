# Palhetas

## PT
Esta App destina-se a trazer o conhecido Blog "Palhetas na Foz" para dispositivos móveis.

Qualquer dispositivo com `Android 5.0`, ou superior, pode instalar a App, através do ficheiro de instalação, encontrado na página de [Lançamentos](https://github.com/danfq/palhetas/releases).

## ENG
This App serves as a way to bring the known "Palhetas na Foz" Blog to mobile devices.

Any device running `Android 5.0`, or above, can install the App, using the installation file, provided in the [Releases](https://github.com/danfq/palhetas/releases) page.
